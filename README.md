Калькулятор для Raspberry Pi OS (интерфейс в стиле macOS)

Описание
- Приложение-калькулятор на Python/Tkinter, вдохновлённое стандартным калькулятором macOS (обычный режим).
- Не требует сторонних библиотек — Tkinter поставляется с Python в Raspberry Pi OS.

Возможности
- Макет 4×5, кнопка «0» занимает две колонки.
- Кнопки функций (AC/C, +/−, %), операций (+, −, ×, ÷), цифры и «=».
- Поведение, близкое к macOS: процент от аккумулятора при активной операции, повторное «=», C/AC, переключение знака.
- Поддержка клавиатуры: 0–9, . , + - * /, Enter/Return (=), Esc (AC/C), Backspace (⌫).

Запуск
1) Убедитесь, что установлен Python 3 (обычно предустановлен в Raspberry Pi OS).
2) Запустите приложение:

   python3 main.py

Примечания
- Окно адаптивно, можно изменять размер.
- Цветовая схема и поведение стремятся быть близкими к macOS, но могут не совпадать на 100%.

Структура
- main.py — основной файл приложения.

Логирование
- Приложение ведёт лог в консоль и в файл calculator.log в корневой папке проекта.
- Уровень логирования можно задать через переменную окружения CALC_LOG_LEVEL.
  Примеры:

    CALC_LOG_LEVEL=DEBUG python3 main.py

- По умолчанию уровень INFO. Для файла используется ротация (до ~500 КБ, 2 резервные копии).
